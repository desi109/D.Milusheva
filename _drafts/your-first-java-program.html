What is OOP and which are its basic concepts?

When we say that Java is an object-oriented programming language we mean that Java is a language in which programs are considered as a collection of objects. Each object is nothing but an instance of a class. There are four principle concepts upon which object-oriented design and programming rest. They are:

Abstraction - refers to the act of representing essential features without including background details or explanations. It shows only the necessary details for an object, not its inner details. Example: A car in itself is a well-defined object, which is composed of several other smaller objects like a gearing system, steering mechanism, engine, which again have their own subsystems. But for humans car is one single object, which can be managed by the help of its subsystems, even if their inner details are unknown.

Polymorphism - can be briefly described as "one interface, many implementations". But if you do not know what is an interface, let's say that polymorphism is a characteristic of being able to assign different meaning or usage to something in different contexts - specifically, to allow an entity such as a variable, a function, or an object to have more than one form.
Example: If we have class Animals. For this example, animals can be cats or dogs. Animals make sounds but the sound of the cat is "meow" and the sound of the dog is "bark". The two sounds are different but they are still sound and they are made from the animals - cat and dog.

Inheritance - it is a concept where one class shares the structure, properties of object and behavior defined in another class. A class that is inherited is called a superclass and the class that does the inheriting is called a subclass.

Encapsulation - is a technique used for hiding the properties and behavior of an object and allowing outside access only as appropriate. It prevents other objects from directly altering or accessing the properties or methods of the encapsulated objects. Levels of encapsulation are public, protected and private (they are called access modifiers). We will discuss them later.

What is Class?

A class is a template from which individual objects are created in the memory. A class can contain fields and methods to describe the behavior of an object.
